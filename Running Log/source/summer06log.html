Weekly Mileage 8/12-8/19: 65
Saturday, August 19th
Workout: run Minutes: 72:30 Miles: 9
Comments: Ran Broken Hills all the way south to UCSD, hitting Guy Fleming on the way down. Logic would have normally told me to run north this morning and let my legs recover on the flat a bit, but I actually like Broken Hills, and I figure that it wouldn't kill me to run one more hill before I leave tomorrow and run essentially flat runs for the last nine months of my running career. I really can't believe that I'm coming up on my senior season, and that practice starts Monday. I'm pumped to see the team and everyone again though.
